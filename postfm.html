Food Ordering Application

This is a project i started in june 2021 when i finished learning the basics of react native and the basics of api integration using react native

This application uses the Midtrans payment gateway to handle payments and the API of this application is made using Laravel

the stage of making this application starts from the needs analysis stage then continues with the creation of a wireframe then continues with the creation of a UI design and enters the development stage by converting the ui design to code using the react native library

after that it is followed by integration with api starting from authentication integration and then integration with food and order data then the final stage is integration with checkout api.

after the integration is complete proceed with testing after the test is successful the last stage is the application build

Technologies:
  • - Javascript
  • - React
  • - React Native
  • - React Redux
  • - Redux
  • - Redux Thunk
  • - Axios
