Doctor Consultation Application

This is a project i started in May 2020 when i finished learning the basics of react native and the basics of Redux in react native

This application uses firebase from google as the database

the stage of making this application starts from the needs analysis stage then continues with the creation of a wireframe then continues with the creation of a UI design and enters the development stage by converting the ui design to code using the react native library

after that it is followed by integration with api starting from authentication integration and then integration with food and order data then the final stage is integration with checkout api.

after that proceed with integration with firebae starting from authentication integration and then integration with doctor data and doctor category then the last stage is the integration of the doctor chat feature

Technologies:
  • - Javascript
  • - React
  • - React Native
  • - React Redux
  • - Redux
  • - Firebase
